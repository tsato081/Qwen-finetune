# Detailed Instruction (for JSONL)

```
ニュース報道から犯罪者の情報を構造化抽出し、以下のJSON形式で出力してください。

【基本ルール】
1. 「逮捕」「容疑者」「被告」など、明確な犯罪者のみ抽出対象
2. 複数犯罪者がいる場合、それぞれを独立したオブジェクトとして配列に追加
3. 犯罪者がいない場合、persons は空配列 []
4. 記事に書かれていない情報は推測・補完禁止。未記載は空文字 ""
5. JSON形式のみ出力。その他のテキストは一切出力禁止

【persons（犯罪者の配列）】
※ 複数犯罪者がいる場合はそれぞれオブジェクトとして記載

- name（必須）: 人物のフルネーム（姓名両方必須）
  →被害者や容疑者でない人物は除外
  →実在しない、架空の場合も記載

- is_real: 実在する人物なら true、架空・偽名なら false

- is_offender: 常に true（犯罪者のみ対象）

- age: 年齢（数字のみ、例："45"）
  →記事に「45歳」と明記 → "45"
  →記事に「50代」などざっくり記述 → 空文字 ""
  →不明の場合 → 空文字 ""

- position: 役職名のみ（社長、役員、元役員、組幹部など）
  【重要な除外】職業情報は除外
  →「会社員」「警備員」「セラピスト」など NG
  →「従業員」「元従業員」「派遣社員」「アルバイト」（職階）OK
  →「無職」の場合は "無職" と記載
  →報道時点での役職のみ（過去のポジションは除外）
  →不明な場合は空文字 ""

- affiliation: 報道時点での所属組織名（現在所属のみ）
  【重要な除外】
  →過去の所属は除外
  →学校は除外
  →無職・職業不詳の場合は空文字 ""
  →「〇〇建設会社」など一般名詞は空文字 ""（「ABC建設」など固有名詞のみ）
  →不明な場合は空文字 ""

- address: 人物の住所（都道府県/市/区/丁目レベル）
  【整形ルール】
  →建物名・部屋番号などの詳細は除外
  →「同市」「同区」は直前の記述に展開・置換
  →例）「東京都渋谷区の自宅」→「東京都渋谷区」
  →記事に明記されていない場合は空文字 ""（推測禁止）

- category: 以下のいずれか
  →"法令違反"（詐欺、暴行、窃盗など犯罪）
  →"PEPS"（著名人スキャンダル等）
  →"破産"（個人破産など）
  →"その他"（分類不明）
  →判定できない場合は空文字 ""

- act: 犯罪名（実在する法的名称）
  →「詐欺」「暴行」「窃盗」など
  →複数ある場合はカンマ区切り（例："詐欺、横領"）
  →基本的に「〇〇の疑い/容疑で」という記述から抽出
  →不明な場合は空文字 ""

【occured_locations（犯行地点の住所配列）】
【重要ルール】
- 対面で直接行われた犯罪のみ対象（加害者と被害者が現場で会う）
  →電話詐欺の「被害者宅」は含めない
- 住所が記事に明示されている場合のみ抽出
- 抽出対象は3タイプ：
  ①実行場所（窃盗・暴行などの発生地点）
  ②計画/準備場所（アジト・拠点等）
  ③発覚/検挙場所（被害申告場所・逮捕場所・証拠発見場所等）
- 住所の整形：都道府県/市/区/丁目を含む表記のみ
  建物名・部屋番号などの詳細は除外
  「同市/同区」は展開・置換
- 記事に住所が明記されていない、または推測が必要な場合は空配列 []

【police_departments（関連警察署の配列）】
【重要ルール】
- 「福岡県警小倉南署」「警視庁〇〇署」など、「〇〇署」の形式
- 「〇〇県警」のみの記述は除外（空配列で返す）
- 記事に警察署が明記されていない場合は空配列 []

【出力形式（必須）】
{
  "persons": [
    {
      "name": "...",
      "is_real": true/false,
      "is_offender": true,
      "age": "...",
      "position": "...",
      "affiliation": "...",
      "address": "...",
      "category": "...",
      "act": "..."
    }
  ],
  "occured_locations": ["...", "..."],
  "police_departments": ["...", "..."]
}

【出力例】
入力：「東京のIT企業ABC社の山田太郎社長（45）が、詐欺容疑で逮捕された。容疑者は渋谷のオフィスで被害者と対面し、偽の契約書を渡したという。渋谷警察署が捜査している。」

出力：
{
  "persons": [
    {
      "name": "山田太郎",
      "is_real": true,
      "is_offender": true,
      "age": "45",
      "position": "社長",
      "affiliation": "ABC社",
      "address": "",
      "category": "法令違反",
      "act": "詐欺"
    }
  ],
  "occured_locations": ["東京都渋谷区"],
  "police_departments": ["渋谷警察署"]
}
```

## チェックリスト（task-explanation.md との照合）

✓ 人物名：犯罪者のみ（被害者除外）
✓ 年齢：intのみ、ざっくり記述は除外
✓ 住所：細かく、「同市/同区」展開ルール
✓ 所属組織：現在所属のみ、学校・無職は除外
✓ 役職名：職階のみ、職業は除外、「無職」は記載
✓ 犯罪名：複数対応、「容疑で」から抽出
✓ 犯行地：対面のみ、3タイプ、電話詐欺被害者宅除外、推測禁止
✓ 警察署：「〇〇署」形式、「県警」のみは除外
✓ 複数犯罪者：独立したオブジェクト
✓ 犯罪者なし：persons=[]
✓ JSON形式のみ出力
✓ 推測・補完禁止
